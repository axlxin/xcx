<p><strong>Overview:</strong></p>
<li>CC2530 Eval Kit is a ZigBee development/evaluation kit designed for CC2530F256, consists of the mother board ZB502 and the ZigBee module Core2530.</li>
<p><strong>ZB502 Features:</strong></p>
<li>Onboard battery holder, multi power supplies are available</li>
<li> All the IOs are accessible for easy expansion<br>
<li>Integrates CP2102 for debugging</li>
<li>LEDs and user keys</li>
<p><strong>Specifications feature:</strong></p>
<li>ZigBee module connector: for easily connecting ZigBee modules</li>
  
  <li>I/Os interface</li>
   <li>USB interface</li>
   <li>UART/SPI interface</li>
   <li>ONE-WIRE interface</li>
  <li> Debugging interface</li>
  <li> 5V/3.3V power input/output: usually used as power output, also common-grounding with other user board</li>
  <li> CP2102</li>
  <li> AMS1117-3.3</li>
  <li>Power switch</li>
  <li>Serial port RX/TX indicator</li>
  <li> Power indicator</li>
  <li> User LEDs</li>
  <li> User keys</li>
  <li>Reset button</li>
  <li> UART0 jumper</li>
