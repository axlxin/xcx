Descriptions:

This DC-DC power module can be used on:
- Lipo battery charger project
- High power LED light driver, this module can keep the output power stable.
- Other projects that need constant current and voltage power supply.

Conversion efficiency: 92% (highest)
Switching frequency: 150 KHZ
